# GraphQL

[toc]

GraphQL 只是一种规范，[官网](https://graphql.org/code/) 提供了不同平台的具体实现，它的实现包括：服务端、客户端、相关工具。

- 服务端：GraphQL.js（官网提供的 js 实现）、apollo-server（第三方实现，功能更加强大）、express-graphql（GraphQL.js 整合到 express 框架）
- 客户端：apollo client、graphql-request 等等，他们只是对 ajax 的封装，使得更加适合 GraphQL；

## 使用

### 服务端

- [GraphQL.js](https://graphql.org/graphql-js/) 最简单的使用分为三步：

```js
const {graphql,buildSchema} = require('graphql')

// 1.构建 schema：Query（查询）
const schema = buildSchema(`
    type Query {
        name:String,
        age:Int
    }
`)

// 2.定义 schema 的 resolver（查询的时候值是什么）
const root = {
    // 查询 name 时会调用当前函数，返回值就是结果（需要符合 schema 类型）
    name(){
        return 'xm'
    },
    age(){
        return 12
    }
}

// 3.查询使用
graphql(schema,'{name,age}',root).then(res=>{
    console.log(res.data.name,res.data.age) // xm 12
})
```

- [express 使用](https://graphql.org/graphql-js/running-an-express-graphql-server/)：

```js
 const {graphql,buildSchema} = require('graphql')
const  express =require('express')
const {graphqlHTTP} = require('express-graphql')


// 1.构建 schema：Query（查询）
const schema = buildSchema(`
    type Query {
        name:String,
        age:Int
    }
`)

// 2.定义 schema 的 resolver（查询的时候值是什么）
const root = {
    // 查询 name 时会调用当前函数，返回值就是结果（需要符合 schema 类型）
    name(){
        return 'xm'
    },
    age(){
        return 12
    }
}

// 3.创建 express 实例、挂载中间件
const app = express()
app.use('/graphql',graphqlHTTP({
    schema,
    rootValue:root,
    graphiql:true //开启 IDE 工具
}))
app.listen(3000)

//  浏览器输入：http://localhost:3000/graphql
```

## graphql 标准

每个 graphql 都会定义一套类型用于描述从服务可能查询到的数据，每当查询到来服务器会根据 schema 先验证再查询。

- Query 类型：一种特殊的对象类型，是查询的入口，另外 Query 只能定义一次；

    ```js
    const schema = buildSchema(`
        不能作为查询的入口
        type User{
            name:String
        }
        查询的入口
        type Query {
            name:String,
            age:Int
        }
        Query 只能定义一次
        type Query {
            name:String,
            age:Int
        }
    `)
    ```

- 标量类型（基本类型）：Int（32 位整数）、Float（双精度浮点）、String（utf-8 字符串）、Boolean（true/false）、ID（唯一标识符，字符串）

- 对象类型：

    ```js
    const schema = buildSchema(`
        type User{
            name:String,
            age:Int
        }
        type Query {
            user: User
        }
    `)
    const root = {
        user(){
            return {
                age: 12,
                name:'xm'
            }
        }
    }
    ```

- 数组类型：

    ```js
        const schema = buildSchema(`
            type User{
                name:String,
                age:Int,
                hobbies:[String]
            }
            type Query {
                user: User
            }
        `)
        const root = {
            user(){
                return {
                    age: 12,
                    name:'xm',
                    hobbies:['唱','跳','rap']
                }
            }
        }
    ```

- 非空类型：默认可以返回 null，如果不能为空在后面加 ``!``，比如 ``name:String!,``，``hobbies:[String!]!`` 表示 hobbies 不能为 null，里面的元素也不能为 null（可以返回 []）；

## 示例

### 服务端 CRUD

- 查询：

```js
const students = [
    { id: '1', name: 'xm', age: 13, hobbies: ['画画', '读书'] },
    { id: '2', name: 'cxk', age: 15, hobbies: ['唱', '跳', 'rap'] }
]

const schema = buildSchema(`
    type Student{
        id:ID!
        name:String!
        hobbies:[String!]!
    }
    type Query {
        students:[Student]
        student(id:ID!):Student
    }
`)
const root = {
    // 查询所有
    students(){
        return students
    },
    // 查询单个
    student({id}){
        return students.find(item=>item.id === id)
    }
}

```

- 添加：

```js
    const {v4:uuidv4} = require('uuid')
    const students = [
        { id: '1', name: 'xm', age: 13, hobbies: ['画画', '读书'] },
        { id: '2', name: 'cxk', age: 15, hobbies: ['唱', '跳', 'rap'] }
    ]
    // 输入参数必须通过 input 定义
    const schema = buildSchema(`
        input InputStu{
            name:String!
            age:Int!
            hobbies:[String!]
        }
        type Mutation {
            create(inputStu:InputStu):Student!
        }
    `)
    const root = {
        // 新增
        create({inputStu}){
            inputStu.id = uuidv4()
            students.push(inputStu)
            return inputStu
        }
    }
```

- 修改：

```js
const students = [
    { id: '1', name: 'xm', age: 13, hobbies: ['画画', '读书'] },
    { id: '2', name: 'cxk', age: 15, hobbies: ['唱', '跳', 'rap'] }
]
const schema = buildSchema(`
    input UpdateStuInput{
        name:String!
        age:Int!
    }
    type Mutation {
        update(id:ID!,stu:UpdateStuInput):Student
    }
`)
const root = {
    // 修改
    update({id,stu}){
        const student = students.find(student => student.id === id)
        if(student){
            student.name = stu.name
            student.age = stu.age
        }
        return student
    }
}
```

- 删除：

```js
    const {v4:uuidv4} = require('uuid')
    const students = [
        { id: '1', name: 'xm', age: 13, hobbies: ['画画', '读书'] },
        { id: '2', name: 'cxk', age: 15, hobbies: ['唱', '跳', 'rap'] }
    ]

    const schema = buildSchema(`
        type DeletionStatus{
            success: Boolean
        }
        type Mutation {
            create(inputStu:InputStu):Student!
            update(id:ID!,stu:UpdateStuInput):Student
            del(id:ID!):DeletionStatus
        }
    `)
    const root = {
        // 删除
        del({id}){
            const idnex = students.findIndex(student => student.id === id)
            students.splice(idnex,1)
            return {
                success:true
            }
        }
    }
```

- 最终代码：

```js
const {buildSchema } = require('graphql')
const express = require('express')
const { graphqlHTTP } = require('express-graphql')
const {v4:uuidv4} = require('uuid')
const cors = require('cors')

const students = [
    { id: '1', name: 'xm', age: 13, hobbies: ['画画', '读书'] },
    { id: '2', name: 'cxk', age: 15, hobbies: ['唱', '跳', 'rap'] }
]
const schema = buildSchema(`
    type Student{
        id:ID!
        name:String!
        hobbies:[String!]!
    }
    type Query {
        students:[Student]
        student(id:ID!):Student
    }
    input InputStu{
        name:String!
        age:Int!
        hobbies:[String!]
    }
    input UpdateStuInput{
        name:String!
        age:Int!
    }
    type DeletionStatus{
        success: Boolean
    }
    type Mutation {
        create(inputStu:InputStu):Student!
        update(id:ID!,stu:UpdateStuInput):Student
        del(id:ID!):DeletionStatus
    }
`)
const root = {
    // 查询所有
    students(){
        return students
    },
    // 查询单个
    student({id}){
        return students.find(item=>item.id === id)
    },
    // 新增
    create({inputStu}){
       inputStu.id = uuidv4()
       students.push(inputStu)
       return inputStu
    },
    // 修改
    update({id,stu}){
        const student = students.find(student => student.id === id)
        if(student){
            student.name = stu.name
            student.age = stu.age
        }
        return student
    },
    // 删除
    del({id}){
        const idnex = students.findIndex(student => student.id === id)
        students.splice(idnex,1)
        return {
            success:true
        }
    }
}

const app = express()
app.use(cors())
app.use('/graphql', graphqlHTTP({
    schema,
    rootValue: root,
    graphiql: true
}))
app.listen(3000)

```

### apollo

[Apollo](https://www.apollographql.com/docs/intro/platform/) 提供了符合 GraphQL 规范的服务端和客户端，可以快捷完成 GraphQL 开发。

```js
const { ApolloServer, gql } = require('apollo-server')
// 可以安装 Apollo GraphQL 插件用于语法高亮
const typeDefs = gql`
    type User {
        name: String
        age: Int
    }
    type Query {
        user:User
    }
`
const user = {
    name: '小明',
    age: 13
}
const resolvers = {
    // 所有 query 入口
    Query: {
        user: () => user
    }
}
const server = new ApolloServer({ typeDefs, resolvers })
server.listen().then(({ url }) => {
    console.log(`🚀  Server ready at ${url}`)
})
```

可以使用 [middleware](https://www.apollographql.com/docs/apollo-server/v2/integrations/middleware/) 集成到其它框架。

如果需要连接 rest/db 等等数据源，可以在 npm 搜索 ``apollo datasource``
